# PoppyBloom-Precipitation-AntelopeValley
UC Merced ES 252 Remote Sensing of The Environment Class Project
Tracking Lag Time Between Precipitation Events and Peak Poppy Blooms in Antelope Valley

This repository contains the Google Earth Engine (GEE) code, points for area of interest, training points for classifier, data collected, and maps.
Raw data is in the Data directory

Sentinel 2 tif files are in this google drive folder due to file size constraints: https://drive.google.com/drive/folders/10Mm7PqbpTNvpI1KXnlt4_HMAfbZfg_ld?usp=drive_link
The google drive will contain the same content provided in this repo.
